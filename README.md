# Andrew Jacobs' Obelisk Website

Andrew Jacobs [passed away](http://forum.6502.org/viewtopic.php?t=6466) on January 8, 2021.
This is a copy of Andrew's website, downloaded on February 2, 2021 before it went down.  It
was hosted at `www.obelisk.me.uk` and before that `www.obelisk.demon.co.uk`.  

Changes have been made for 6502.org: [http://6502.org/users/obelisk/](http://6502.org/users/obelisk/).  The use of HTML frames has been removed to allow pages to be bookmarked.  The markup has also been updated to make the layout responsive (i.e. more usable on mobile).  The unmodified website has been [preserved](https://github.com/6502org/obelisk/commit/91510c224575c2585eb3af8817c16a75b25615f6) in this Git repository.

See also Andrew's [GitHub repositories](https://github.com/andrew-jacobs).
